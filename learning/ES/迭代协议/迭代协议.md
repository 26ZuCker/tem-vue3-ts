## 迭代协议

参考：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide/Iterators_and_generators

共有六种迭代协议，但只建议使用前四种即可：

1. for
2. for...in
3. for...of
4. for await...of：用于并发，具体参考下方 async
5. do...while
6. while
7. for each...in

推荐写法:

```js
for (var i = 0, max = myarray.length; i < max; i++) {
  // 使用myarray[i]做点什么
}
```
