#### FP

js 内存在以下两种方式定义函数

```js
//正常
a();
//declaration
function a() {}

//以下三者均undefined
c();
d();
e();
//expression
var c = function() {};
//以下本质也是expression
var d = function e() {};
```

二者区别：

- expression 为匿名函数即其 name 属性为空字符串，相应的 declaration 为函数名即()前 function 关键字后的字符串
- expression 不存在函数提前即如以上例子，除非提前声明变量再赋值如`var c; c = function() {};`
