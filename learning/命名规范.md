### 命名

- 当前组件内属性正常驼峰`aB`
- 需要通过 api 获得的变量`c`或`c_c`，直接调用 api 的方法`get_c`
- 私有变量即函数内参数`_d`
- 工具函数，插件包括 vuex 或辅助类实例中属性和方法`$e`
