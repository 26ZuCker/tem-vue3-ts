#### AOP & COP

分别为面向切面和面向组合，二者本质思维都是不断 compose

AOP 基于抽象类进行横向扩充，对比于 OOP 的纵向扩展，相当于 TS 的装饰器

COP 完全一个新的类，相当于 Composition API 通过多个函数挂载而形成一个新的构造函数，得以选择性的继承，而不是传统的继承，因为后者不能有选择性的继承而必须拥有父类的所有属性和方法，当然对于 JavaScript 在内存中是共享
